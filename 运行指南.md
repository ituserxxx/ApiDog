# ApiGog 运行指南

## 项目结构

```
ApiGog/
├── server/          # Flask后端
│   ├── app.py       # 主应用文件
│   ├── requirements.txt  # Python依赖
│   ├── .env         # 环境变量
│   └── config.py    # 配置文件
└── web/             # Vue前端
    ├── package.json # Node.js依赖
    ├── vite.config.js # Vite配置
    ├── index.html   # HTML入口
    └── src/         # 源码目录
        ├── main.js  # Vue入口
        └── App.vue  # 根组件
```

## 后端启动步骤

1. 进入server目录：
   ```bash
   cd server
   ```

2. 创建虚拟环境（推荐）：
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   # 或
   venv\Scripts\activate     # Windows
   ```

3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

4. 启动服务：
   ```bash
   python app.py
   ```

   后端服务将在 http://localhost:5000 运行

## 前端启动步骤

1. 进入web目录：
   ```bash
   cd web
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

   前端应用将在 http://localhost:3000 运行

## 功能验证

- 后端健康检查：http://localhost:5000/api/health
- 前端页面：http://localhost:3000
- API代理：前端通过 `/api` 路径代理到后端

## 技术栈

- 后端：Flask 3.0 + Flask-CORS
- 前端：Vue 3 + Vite + Axios
- 代理：Vite开发服务器代理API请求到Flask
